取消监听加速度数据事件，参数为空，则取消所有的事件监听。

## 类型

```tsx
(callback: (...args: any[]) => any) => void
```

## 参数

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| callback | `(...args: any[]) => any` | 加速度数据事件的回调函数 |
