创建激励视频广告组件。

## 类型

```tsx
(option: Option) => RewardedVideoAd
```

## 参数

### Option

| Name | Type | Default | Description |
| --- | --- | :---: | --- |
| adUnitId | `string` |  | 广告单元 id |
| multiton | `boolean` | `false` | 是否启用多例模式 |
